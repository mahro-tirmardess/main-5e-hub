# main-5e-hub

本仓库用于归档和发布多个独立的 HTML 页面。这些页面不针对任何特定域名或网站，仅作为静态页面的集中存储与分发用途。

## 项目简介

main-5e-hub 是一个轻量级的页面归档仓库，主要存放各种独立编写的 HTML 文件。每个文件均为自包含的页面，可直接在浏览器中打开运行，无需额外依赖或构建步骤。

## 目录说明

仓库根目录下直接存放 HTML 文件，暂未划分子目录。所有页面均为独立文件，文件名即页面标识。

- 根目录：存放所有 HTML 页面文件
- 无子文件夹结构，便于快速浏览与引用

## 页面归档说明

本仓库中的页面均以“单一 HTML 文件”形式归档，不依赖外部资源（如 CDN 样式库或脚本库）。部分页面可能包含内嵌的 CSS 与 JavaScript，以保证离线可用性。

归档的页面类型包括但不限于：

- 信息展示页
- 工具类页面
- 实验性页面
- 其他静态 HTML 内容

每个页面均经过基本测试，确保在主流浏览器中正常渲染。

## 维护说明

- 本仓库不定期更新，新增页面会直接提交至主分支
- 不保证所有页面长期可用或内容不变
- 如需引用某个页面，建议直接使用文件链接
- 不提供 issue 或 pull request 响应保证

## 使用方式

1. 直接点击仓库中的 `.html` 文件，选择“查看原始文件”或通过 GitHub Pages 访问
2. 将文件下载到本地，双击在浏览器中打开
3. 可自由复制、修改、分发（请遵守开源协议）
